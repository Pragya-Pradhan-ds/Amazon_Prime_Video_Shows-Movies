📊 Amazon Prime Video Content Analysis

This project analyzes the content available on Amazon Prime Video using Exploratory Data Analysis (EDA) and an interactive dashboard built in Microsoft Power BI.

The objective of this project is to explore trends in movies and TV shows, understand viewer-oriented attributes such as ratings and genres, and identify patterns that can help streaming platforms make better content and licensing decisions.

📂 Dataset Information

Dataset Name: Amazon Prime TV Shows and Movies Dataset

Description:
The dataset contains detailed information about movies and TV shows available on Amazon Prime Video, including title information, ratings, genres, release year, runtime, and production countries.

📊 Key Attributes
Attribute	Description
title	Name of the movie or TV show
type	Category of content (Movie or Show)
release_year	Year of release
age_certification	Content rating (e.g., PG, R, PG-13)
runtime	Duration of the content in minutes
genres	Genre category of the title
production_countries	Country where the content was produced
imdb_score	IMDb rating of the title
imdb_votes	Number of IMDb votes
role	Indicates whether the person is an Actor or Director

📈 Dashboard Insights (Power BI)

The Power BI dashboard helps answer several key business questions:

⭐ Top Rated Titles

Identifies movies and TV shows with the highest IMDb scores, helping understand high-performing content.

📅 Content Release Trends

Shows how the number of titles released on Amazon Prime has evolved over the years.

🎬 Content Type Distribution

Analyzes the proportion of Movies vs TV Shows available on the platform.

🔞 Age Certification Analysis

Examines the distribution of content ratings such as PG, PG-13, R, etc.

⏱ Runtime and Ratings Comparison

Compares the average runtime and IMDb score for movies and TV shows.

🌍 Genre and Country Distribution

Highlights the most common genres and production countries in the dataset.
